# Contributing

Thanks for your interest in contributing! This project aims to be practical, strict, and free of hidden fallbacks.

## Development

- Prereqs: latest stable Rust, `cargo`.
- Tests:

```bash
cargo test -q
```

## Commit and PR guidelines

- Keep changes focused; add tests for public behavior.
- Avoid introducing hidden defaults or fallback behavior.
- Prefer explicit errors over silent degradation.
